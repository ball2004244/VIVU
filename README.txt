This game about a guy who wanted to travel around the world in order to gather all the badges for his collection.

The game contains: 
+ protagonist: npc(maybe)
+ antagonist(enemies): creeps, mega creeps, boss, quiz,...
+ traps: spike, rock,...
+ biomes (background): town, hill, snow mountain, lava, ice, boss room
+ main player system: weapon, clothings, skills, 
+ mechanism: jump, run, attack, damage, hitbox 
+ shop: buy/sell (potions, clothing, weapon, fishing rod,...)
